# Money Matters: Budget & Thrive

Money Matters is a personal finance dashboard designed to help users track expenses, visualize spending habits, and stay in control of their financial goals.

## 🌐 Live Demo
https://virginiamarc.com/mmbt/

## 🛠 Tech Stack
- HTML / CSS / JavaScript
- Responsive UI
- Local storage for data persistence

## ✨ Features
- Add, edit, and delete transactions
- Track income vs. expenses
- Visual spending summaries
- Clean, modern interface

## 📁 Project Purpose
This project was built to demonstrate practical budgeting tools with a simple, intuitive user experience.
